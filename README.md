# NIMA Project

Sistema de apoyo y orientación para mujeres en situaciones de violencia de género.

## Estructura del Proyecto
```
Nima-project/
├── frontend/          # React + Vite
├── backend/           # Node.js + Express
└── README.md
```

## Instalación

### Frontend
```bash
cd frontend
pnpm install
pnpm dev
```

### Backend
```bash
cd backend
pnpm install
pnpm dev
```

## Stack

### Frontend
- React 18
- Vite
- Tailwind CSS
- React Router DOM
- Lucide React (iconos)

### Backend
- Node.js
- Express
- MongoDB + Mongoose
- JWT (próximamente)
